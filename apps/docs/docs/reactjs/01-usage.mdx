# Basic usage

To use the libraries packages, you need to follow this steps:


## Function Component

1. Use the component in your Component page as follow

```js
import '@fazio/ipfs-upload-button';

export function App() {
  const token = 'WEB3STORAGE_APIKEY';

  return (
    <>
      <div/>
      <web3-upload-btn token={token}></web3-upload-btn>
      <div />
    </>
  );
}
export default App;
```
