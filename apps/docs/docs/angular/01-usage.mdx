# Basic usage

To use the libraries packages, you need to follow this steps:


## Standalone Component
1. Use the component in your Typescript Component page as follow
```js
import { Component, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';
import '@fazio/ipfs-upload-button';

@Component({
  standalone: true,
  schemas: [CUSTOM_ELEMENTS_SCHEMA],
  selector: 'app-root',
  template: `
  <div>
    <web3-upload-btn token="{{token}}"></web3-upload-btn>
  </div>
  `
})
export class AppComponent {
  token = 'WEB3STORAGE_APIKEY'
}
```
